A full-featured e-commerce platform for purchasing premium nuts and seeds. Built with Django and designed with all essential modern e-commerce functionalities including wishlist, cart, orders, returns, reviews, and secure checkout.


----------------------User Features----------------------

User registration, login, and Google OAuth login

Browse products by category

Search product and filtering

Product detail page with images & descriptions

Wishlist management

Add to cart, update quantity, remove items

Checkout with COD, Razorpay, and Wallet payments

Order placement & order history

Order tracking

Return order functionality

Product reviews & ratings

User profile & address management

----------------------Admin Features----------------------

Admin Dashboard, Data analysis

Top selling products, Top selling categories

Add, edit, delete products

Manage categories

View and process orders

Manage returns & refunds

Wallet management

----------------------Tech Stack----------------------

Frontend: HTML, Tailwindcss, Bootstrap, JavaScript

Backend: Django

Database: PostgreSQL

Authentication: Django-Allauth (Email + Google Login)

Payments: Razorpay, COD, Wallet system

Deployment: Gunicorn + Nginx on AWS EC2
